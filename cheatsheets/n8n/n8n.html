Instalación de n8n (self-hosted) con contenedores en Docker (Podman)

En este documento se explica el procedimiento para la instalación de n8n usando contenedores en docker.

Última actualización: 31 de Marzo del 2026

1. Instalación de dependencias elementales

Necesitamos instalar el contenedor de podman

user@linux-server:~

sudo yum install -y podman

2. Crear el contenedor de Docker

user@linux-server:~

podman run -d \

--name n8n_final \

-p 5678:5678 \

-v n8n_data:/home/node/.n8n:Z \

-e N8N_PROTOCOL=https \

-e N8N_HOST=emiguzman.ddns.net \

-e N8N_PATH=/n8n/ \

-e WEBHOOK_URL=https://emiguzman.ddns.net/n8n/ \

docker.io/n8nio/n8n
